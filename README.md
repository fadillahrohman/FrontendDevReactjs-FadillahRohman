# Restaurants Website

A modern restaurant website built with React, TypeScript, and Vite.  
This project provides a fast and responsive UI for exploring restaurant data.

---

## Tech Stack

- React — ^19.2.4  
- TypeScript — ~6.0.2  
- Vite — ^8.0.4  
- Tailwind CSS — ^4.2.2  
- React Router DOM — ^7.14.0  

---

## Environment Requirements

Make sure you have installed:

- **Node.js** ≥ 18 (Recommended: 18 / 20 LTS)
- **npm** or **yarn**

Check your version:

```bash
node -v
npm -v
````

---

## Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/fadillahrohman/FrontendDevReactjs-FadillahRohman.git
cd restaurants
```

### 2. Install Dependencies

Using **npm**:

```bash
npm install
```

Or using **yarn**:

```bash
yarn install
```

### 3. Run Development Server

Using **npm**:

```bash
npm run dev
```

Using **yarn**:

```bash
yarn dev
```

App will run on:

```
http://localhost:5173
```

---

## Build Project

```bash
npm run build
```

Or:

```bash
yarn build
```
